Description:

Alert is an Android Application which can be used as Emergency tool to Alert Peoples close to you when you are in danger or feeling unsafe.

Introduction Video

Demo Video

It has three ways to Alert:

Medical Emergency:

It takes users current location and finds nearest hospital to the user and make a direct call for you with just a single touch.

Alert Emergency Contacts:

At start Alert asks you for 5 Emergency contacts (5 necessary) and then if you press Alert button it messages the peoples with users current location with latitude and longitude and complete address in string format and users pre typed custom message.

Police Emergency:

It takes users current location and finds nearest Police Station to the user and make a direct call for you with just a single touch and if it fails to find the station number it will call 100.

Alert can be used in Women's safety, in road accidents and ultimately to save someone's life.
